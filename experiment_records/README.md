# Experiment Records

This directory is intentionally reset for the clean 5-split rerun.

- `comparisons/` is reserved for small cross-method CSV/JSON comparison tables.
- `reports/` is reserved for paper-facing Markdown summaries and run plans.
- Raw models, checkpoints, configs, and logs stay under `experiments/<dataset>/<method>/runs/`.
- Method-local split results stay under `experiments/<dataset>/<method>/records/split_results.*`.

The current rerun plan is generated by:

```powershell
F:\Anaconda\envs\pytorch\python.exe tools\run_ordered_5split_experiments.py --num_splits 5
```

For the same dataset and method, the runner continues from the next split after
the latest completed split in the existing run folder.  After each method
finishes, the run folder suffix is updated to the cumulative completed split
count, for example `brca_sentence_only_5splits` -> `brca_sentence_only_10splits`.
